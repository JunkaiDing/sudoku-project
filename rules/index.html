How to Play

Sudoku is a logic-based, combinatorial number-placement puzzle. The objective is to fill a grid with digits so that each column, each row, and each of the subgrids that compose the grid contain all of the digits from 1 to 9 (or 1 to 6 for easy mode).

  1. Grid Logic: The game is played on a grid (9x9 or 6x6). You must fill every empty cell with a number.
  2. Row Rule: Each row must contain the numbers 1-9 (or 1-6) exactly once. No repeats allowed!
  3. Column Rule: Each column must contain the numbers 1-9 (or 1-6) exactly once. No repeats allowed!
  4. Subgrid Rule: Each subgrid (3x3 for hard, 2x3 for easy) must contain the numbers 1-9 (or 1-6) exactly once.
  5. No Guessing: Sudoku is a game of logic and reasoning. You shouldn't have to guess; there is always a logical next step.
